# Task Management API

## Features
- User & Task Management
- Indexed queries for performance
- In-memory caching
- Cloud deployment ready

## API Endpoints

### Get Tasks
GET /api/tasks

### Create Task
POST /api/tasks

### Update Task
PUT /api/tasks/:id

## Installation
1. Clone repo
2. npm install
3. Create .env file
4. npm start

## Tech Used
- Node.js
- Express
- MongoDB
